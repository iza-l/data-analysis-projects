# 📊 Controle de Qualidade com Excel

Projeto de análise de dados aplicado ao controle de qualidade em processos industriais, utilizando Excel para organização, tratamento e visualização de dados.

## 📌 Objetivo
Analisar dados de produção simulados, avaliando parâmetros como pH e viscosidade para determinar a conformidade dos lotes, além de identificar possíveis relações entre variáveis do processo e seu impacto na qualidade final do produto.

## 🛠️ Ferramentas utilizadas
- Excel  
- PROCV  
- Tabela dinâmica  
- Gráficos (pizza, coluna e dispersão)  
- Tratamento e organização de dados  

## 📈 Análise
A análise demonstrou que a maioria dos lotes está dentro dos padrões estabelecidos.  
Observa-se que, à medida que a temperatura de produção aumenta, há uma tendência de redução da viscosidade, indicando possível influência das condições térmicas no comportamento do produto.

## 📊 Dashboard
O projeto inclui um dashboard com:
- Indicadores de desempenho (total, aprovados e reprovados)  
- Distribuição dos lotes  
- Comparação de médias  
- Relação entre temperatura e viscosidade  

## 📎 Arquivo
O arquivo Excel contém:
- Base de dados  
- Tabela de parâmetros  
- Análise com tabela dinâmica  
- Dashboard final  

## 🎯 Resultado
Este projeto demonstra habilidades em análise de dados, organização de informações e criação de dashboards no Excel.
